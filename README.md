# RhombixTechnologies_Task04
This repository contains an online food delivery website created as part of the internship task.  

The website is developed using **HTML, CSS, and JavaScript**.  
It is designed to allow users to browse food items, add them to a cart, place orders, and manage content via an admin page. The website is fully **responsive**, working on desktop, tablet, and mobile devices.  

## Project Overview
This online food delivery website includes:

- **Home Page** – Browse food categories and featured items  
- **Cart Page** – View selected items, update quantities, and see total price  
- **Login Page** – User authentication  
- **Admin Page** – Manage food items and orders  
- Fully **responsive design** for desktop, tablet, and mobile  

## Technologies Used
- HTML5  
- CSS3  
- JavaScript  

## Purpose of This Task
The purpose of this task is to demonstrate:

- Building a multi-page website with structured navigation  
- Applying responsive web design concepts  
- Implementing dynamic functionality using JavaScript
